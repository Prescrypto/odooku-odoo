Odooku Odoo

Install Odoo through pip.


# System requirements

# Ubuntu 16

```
sudo apt-get install libpq-dev libxml2-dev libxslt1-dev libldap2-dev libsasl2-dev libssl-dev
```

# OSX

```
brew install postgresql
```
